# Link Analysis Platform

Repository: [ArgosML-tech/link_analysis](https://github.com/ArgosML-tech/link_analysis)

## Purpose

`link-analysis-platform` is an analytical application for graph-based investigation, entity exploration, configurable risk analysis, and report generation.

## Business Problem

Operational source systems are not designed for relational investigation across heterogeneous entities and signals. This application focuses on downstream interpretation, graph navigation, and risk-oriented analysis.

## Core Capabilities

- graph exploration and navigation
- configurable ontology and risk rules
- file ingestion workflows
- report generation
- graph-aware chat and analysis assistance
- LAP-CAP run import into the analysis flow

## Main Patterns

- downstream analytical consumer
- graph-first UI
- interpretation of staged or imported business data
- separation from extraction responsibilities

## Why It Matters in This Collection

This repository broadens the collection beyond CAP-only applications. It shows how CAP-based integration services can feed a richer downstream analytical product.
