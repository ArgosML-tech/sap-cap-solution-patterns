# CAP-MIL

Repository: [ArgosML-tech/cap-mil](https://github.com/ArgosML-tech/cap-mil)

## Purpose

`cap-mil` implements an urgent procurement workflow with reinforced traceability, approvals, procurement follow-up, and closure.

## Business Problem

Urgent purchases are often operationally necessary but weak in governance. The project addresses that by formalizing the lifecycle of emergency procurement cases.

## Core Capabilities

- urgent case submission
- unit approval and procurement review
- linkage to core document references
- execution and completion tracking
- comments, approvals, attachments, and event logs

## Main Patterns

- end-to-end lifecycle modeling
- explicit state transitions with comments and decisions
- audit trail
- Fiori Elements transactional UI

## Why It Matters in This Collection

This project extends the workflow theme beyond simple approval into operational follow-through, making it a useful reference for richer process applications.
